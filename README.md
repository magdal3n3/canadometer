# Canadometer

A checklist-based web app to track iconic Canadian experiences — food, culture, nature, sports, history, politics, and more. Find out how Canadian you really are.

## Built with

- React 18
- Vite
- No external UI libraries — all custom styles

## Running locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

## Deploying

This project is configured for deployment on [Vercel](https://vercel.com). Push to GitHub and connect your repo to Vercel — it handles the rest automatically.
